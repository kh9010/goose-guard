System architecture

Flask serves the control panel and REST API. SoundEngine picks clips using weighted categories plus a recently-played deque so geese don't habituate. Scheduler plans dawn, dusk and midday sessions using astral's sunrise/sunset for the configured location. MotionMonitor polls a PIR sensor and plays bursts when motion mode is on.
